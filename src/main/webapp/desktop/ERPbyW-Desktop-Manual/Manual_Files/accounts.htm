The screen below will appear.

New Account will allow you to define a new account, Delete Account will delete an account, Make Payment will allow you to make a payment.

Payments can only be made from one account to another. Every item must be defined as an account. EG: The electricity company will be an account and that account will receive payments from the Company's Bank Account.

Please see the Payroll section for more information on the Payroll module.

The Invoice, Receipt and Payroll modules post their totals to the Trial Balance report. You can edit this report by clicking on Edit Trial Balance

The Ledger is the Trial Balance report. You must select a range of dates for the transactions you would like to view.
