Clients/Stakeholders

Once you have logged into Maestro, click on File - New Client to define a new client/customer.



To edit a client, click on File - Edit Client. You will be asked to type in the forenames of the client you want to edit. You can type in a few letters of the clients name, but to be more accurate it is safer to type in the forenames of the client because the system might return a client with similar characters in their name as the characters in the client you want to edit's name.

To view a listing of all clients click on File - List Clients.
