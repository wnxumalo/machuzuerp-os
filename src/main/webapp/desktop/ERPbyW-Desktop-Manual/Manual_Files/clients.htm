CUSTOMERS

Click on File - New Client from the main menu or the button.

The screen below will appear.

Enter the customer's details in the fields. Not all fields have to be filled in.

New will allow you to enter in a new client.

Find will allow you to locate a client's record either for viewing or editing.

To edit a client from the main screen, click on File - Edit Client. You will be asked to type in the forenames of the client you want to edit. You can type in a few letters of the clients name, but to be more accurate it is safer to type in the forenames of the client because the system might return a client with similar characters in their name as the characters in the client you want to edit's name.

To view a listing of all clients click on List.

Invoices will show all invoices for the current client.

Amount Due will show the Total Amount Outstanding for the current client.

Enter an Account Limit to set the limit of a client's account. ie: how much a client can purchase on credit/debit
