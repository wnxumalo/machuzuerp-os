ADMINISTRATION

The Administration module is the Control Panel where you can set up the users of your system, the companies, tax, markups etc.

To log into Administration, click on System - Login.



If it is the first time using the program you need to log in as the System Administrator. Type in the administrator username and password in the relevant boxes. The username and password for the Sytem Administrator are:

Username: admin
Password: admin1

The password can be changed for security every now and again in the Administration Module by clicking on File - Change Administrator Password.

Users

Fill in the fields in the User section of Administration.

Mandatory fields for users are the Full Name, Username and Password fields.

The Username and Password defined here will be used to log into the system.

The Administrator should set users up and enable a user's access to modules by clicking/ticking/checking the module in the User section. As shown, all the modules have been selected for the User Wandile Nxumalo.

Companies

Fill in the fields in the Company section of Administration.

Mandatory fields for users are the Name and at least one contact detail.

You can upload your Company Logo to appear on Invoices, Statements etc by clicking the Upload Logo button.

The Tax Rate is the rate which you tax your goods sold by. This rate will be applied to invoices, receipts etc when they are created. The total for the invoices, receipts etc will include the tax on the items. Please see the Invoices section

Once you have defined a user and a company you are ready to log into the Maestro Invoicer.
